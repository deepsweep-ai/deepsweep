# Changelog

All notable changes to DeepSweep will be documented in this file.

The format is based on Keep a Changelog, and this project aims to follow Semantic Versioning.

## [0.1.0] - 2025-12-24

### Added
- Initial public release of the DeepSweep CLI
- Validation for AI assistant configuration surfaces (Cursor, Copilot, Claude Code, Windsurf, MCP)
- Text, JSON, and SARIF output formats
- Detection patterns for prompt injection, MCP poisoning, and related configuration risks
